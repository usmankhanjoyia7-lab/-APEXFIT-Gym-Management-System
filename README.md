## ✨ Overview

**APEXFIT** is an ultra-professional, production-ready gym management and membership booking system designed specifically for fitness centers in Pakistan. This comprehensive platform combines modern web technologies with enterprise-grade functionality to deliver a seamless experience for gym owners, trainers, and members.

Built as a **portfolio-quality project**, APEXFIT demonstrates advanced frontend development skills including responsive design, interactive features, SEO optimization, and user-centric interface design. The system is fully functional, scalable, and ready for real-world deployment.

---

## 🎯 Key Features

### 🏢 **Gym Management**
- **Multi-Session Scheduling:** Morning (6:00 AM - 10:00 AM) and Evening (7:00 PM - 11:00 PM) sessions, 7 days a week
- **Class Management:** Real-time seat availability tracking with visual indicators
- **Trainer Profiles:** Detailed trainer information with specializations and booking capabilities
- **Facility Showcase:** Interactive carousel displaying gym facilities with overlay descriptions

### 💳 **Membership Plans**
- **Three-Tier Pricing Model:** Basic (PKR 3,000), Pro (PKR 5,000), and Elite (PKR 8,000) monthly plans
- **Feature Comparison:** Clear differentiation between plan benefits and inclusions
- **Flexible Membership:** No long-term contracts, cancel anytime

### 📅 **Advanced Scheduling System**
- **Weekly Class Timetable:** Comprehensive schedule with real-time seat tracking
- **Interactive Booking:** One-click class booking via WhatsApp integration
- **Availability Indicators:** Color-coded seat availability (green for available, red for low availability)
- **Multiple Class Types:** HIIT, Yoga, Strength Training, Cardio, Functional Training, and more

### 🧮 **Health & Fitness Tools**
- **BMI Calculator:** Instant BMI calculation with personalized health recommendations
- **Daily Calorie Calculator:** TDEE (Total Daily Energy Expenditure) calculation with macro recommendations
- **Customizable Inputs:** Gender, age, weight, height, and activity level considerations

### 📱 **Communication & Engagement**
- **WhatsApp Integration:** Direct WhatsApp messaging for inquiries, bookings, and memberships
- **Contact Form:** Comprehensive contact form with plan selection and custom messaging
- **Social Media Links:** Integrated social media connections (WhatsApp, Instagram, Facebook, Email)
- **Live Location Map:** Embedded Google Maps showing exact gym location

### 📚 **Content Management**
- **Blog Section:** Fitness tips, nutrition guides, and training articles
- **FAQ Section:** Comprehensive Q&A covering memberships, hours, policies, and procedures
- **SEO Optimization:** Full meta tags, Open Graph, Twitter Cards, and structured schema markup

### 🎨 **User Experience**
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices
- **Dark Theme:** Modern, eye-friendly dark interface with orange accent colors
- **Smooth Animations:** Fade-up effects, parallax scrolling, and interactive hover states
- **Accessibility:** ARIA labels, semantic HTML, and keyboard navigation support
- **Performance Optimized:** Split JavaScript files for faster loading and better caching

## 💡 Usage Guide

### **For Gym Owners/Managers**

1. **Customize Content**
   - Edit `index.html` to update gym name, contact info, and location
   - Modify `scheduleData` in `script2.js` to update class schedules
   - Update trainer information and images in the Trainers section

2. **Update Pricing**
   - Modify membership prices in the Membership section
   - Update plan features and descriptions

3. **Configure WhatsApp**
   - Replace `WA_NUMBER` in `script1.js` with your WhatsApp number
   - Update WhatsApp message templates as needed

4. **Add Your Content**
   - Update blog articles in the Blog section
   - Add FAQ items specific to your gym
   - Update contact information and opening hours

### **For Members/Users**

1. **View Class Schedule**
   - Click on different days to see available classes
   - Check seat availability in real-time
   - Book classes directly via WhatsApp

2. **Use Health Tools**
   - Calculate your BMI with instant recommendations
   - Calculate daily calorie needs based on activity level

3. **Explore Membership Plans**
   - Compare features across Basic, Pro, and Elite plans
   - Join directly through WhatsApp
